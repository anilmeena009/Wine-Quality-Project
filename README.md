# Wine-Quality-Project
Over view:- The Wine Quality Prediction Project uses machine learning to predict wine quality based on its physicochemical properties like alcohol, pH, and acidity. It involves data preprocessing, exploratory data analysis, and model building. The goal is to identify key factors affecting wine quality and achieve accurate predictions. This project demonstrates the practical application of data science in quality assessment.

Dataset:-
File used: winequality-red.csv
Target: Wine Quality (score)
Features include acidity, sugar, pH, alcohol, etc.

Features:-
Data Cleaning & Preprocessing,
Exploratory Data Analysis (EDA),
Feature Engineering,
Model Training & Evaluation,
Visualization of insights.

Tech Stack:-
Python, 
Pandas,
NumPy,
Matplotlib / Seaborn,
Scikit-learn.

how to use it :-
Open Jupyter Notebook
Run all cells step by step
Check model performance and visualizations
